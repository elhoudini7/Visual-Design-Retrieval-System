# Visual-Design-Retrieval-System
A full-stack computer vision system for analyzing visual designs through layout understanding, style classification, image embeddings, and similarity search.

## Planned Features

- Image Upload
- Color Palette Extraction
- Layout Analysis
- Visual Hierarchy Detection
- Style Detection
- Image Embeddings
- Similarity Search
- Design Scorecard
